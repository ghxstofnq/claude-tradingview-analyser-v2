Visual direction — pure terminal

The chosen aesthetic. The same “active setup” panel that defined the direction — monospace throughout, near-black surface, hairline borders, zero rounding, semantic colour. The chart keeps TradingView's own look; this language applies to every other surface.

Rules

  • Monospace throughout. Uppercase, letter-spaced labels for headers.
  • 1px hairline borders. Zero border-radius. No shadows, no gradients.
  • High density — whitespace groups, it does not decorate.
  • Colour is semantic, never decorative. Buttons are bracketed: [ ACCEPT ].
